# 03 中国玄学系统路线

## 中国术数总架构

中国术数适合按“五大内核”组织：

1. 阴阳五行：五行生克制化、旺相休囚死、十神、六亲。
2. 干支历法：天干地支、节气、月令、旬空、纳音、神煞。
3. 易卦象数：八卦、六十四卦、爻、互变错综、河洛。
4. 九宫方位：洛书、九宫、八门、九星、八神、飞布。
5. 星宿神煞：二十八宿、七政四余、紫微星曜、神将。

星阙应做成“规则可复用”的中国术数底座：干支、节气、真太阳时、农历、神煞、八卦、九宫、二十八宿、十二长生、纳音、六亲、十神都应是共享基础库。

## 八字

### 核心计算

- 四柱：年柱、月柱、日柱、时柱。
- 年柱切换：立春切年、春节切年、子正切日、晚子时等可选。
- 月柱：以节气月为主，明确节/中气规则。
- 时柱：真太阳时、地方平太阳时、标准时可配置。
- 藏干：本气、中气、余气，权重版本。
- 十神：以日主为中心。
- 十二长生：阳顺阴逆、五行长生版本差异。
- 纳音：六十甲子纳音。
- 神煞：天乙、文昌、桃花、驿马、华盖、羊刃、禄神、空亡等，需来源版本。
- 大运：顺逆、起运岁数、节气距离、三天一岁/一天四月等算法。
- 流年、流月、流日、流时。

### 主要流派

| 流派 | 特点 | APP 功能 |
| --- | --- | --- |
| 子平格局 | 月令取格、格局成败、用神忌神 | 格局判定器、成败证据 |
| 旺衰平衡 | 日主强弱、扶抑、通关、调候 | 旺衰评分、五行热力图 |
| 调候派 | 寒暖燥湿、月令气候、病药 | 调候用神解释、季节气候面板 |
| 盲派 | 象法、宫位、宾主、做功、穿刑破害 | 象法断语树、事件定位 |
| 神煞派 | 神煞组合、吉凶象义 | 神煞版本管理、命中证据 |
| 纳音派 | 纳音五行、纳音生克 | 纳音关系网 |
| 格局/财官派 | 财官印食、清浊、成败 | 十神组合识别 |
| 河洛/卦气 | 命卦、数理、卦象 | 八字转卦、河洛图 |

### 顶级功能

- 格局对照：同一八字按多个流派给出不同判定。
- 用神争议面板：显示“扶抑用神、调候用神、格局用神”差异。
- 命例库：可按日主、格局、神煞、事件类型检索。
- 事件反推：用户录入年份事件，系统比较大运/流年/原局触发关系。
- 断语结构化：将“财星坏印”拆成规则、条件、强度、反例。

- 主星：紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军。

- 四化：化禄、化权、化科、化忌。
- 大限、小限、流年、流月、流日。
- 身宫、命主、身主、斗君。

### 主要流派

| 流派 | 特点 | 功能 |
| --- | --- | --- |

## 太乙神数

### 核心计算

- 年计、月计、日计、时计、分计。
- 太乙、文昌、始击、定目、主客大小将、计神、合神。
- 九宫/十六神/五福/八风等。
- 阳遁阴遁、局数。
- 统宗、金镜、淘金歌等不同口径。

### 产品方向

- 太乙更适合用于国运、军事、灾异、天气、宏观趋势、择时。
- 应明确它与个人命理的边界，避免把宏观术法硬套个人小事。
- 当前已有参考实现，应重点做“盘面解释”和“古籍规则引用”。

### 基础

- 二十四节气、朔望月、农历月、闰月。

### 顶级功能
